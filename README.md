# Data-Driven-Machine-Learning-Heart-Disease-Predictor.

## Heart Disease Prediction System

A machine learning-based clinical decision support system built with Django, Scikit-learn and Streamlit for predicting heart disease risk using patient medical data.

## Features
Heart disease prediction

## Machine Learning
Models Used:
- Random Forest
- Decision Tree
- Support Vector Machine (SVM)
- Linear Regression

Data preprocessing:
- Missing value handling
- Duplicate removal
- Outlier treatment (Z-score)
- One-hot encoding
- Feature scaling

## Technologies
Python
Django
Scikit-learn
Pandas
NumPy
Streamlit

## Installation
git clone https://github.com/yourusername/heart-disease-prediction.git
cd heart-disease-prediction
pip install -r requirements.txt

streamlit run app.py
